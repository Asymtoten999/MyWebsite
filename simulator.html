Coin Flip Simulator

Simulate 10,000 coin flips and estimate P(X > threshold)


Summary:

This project explores probability through large-scale coin flips, comparing empirical simulation with analytical approximation using the Central Limit Theorem (CLT). Users can interact with the simulation, see the distribution of outcomes, and understand the relationship between random experiments and theoretical predictions.

🧠 What the project does

  • Simulates many coin flips: each trial flips a fair coin n times (e.g., 10,000) and repeats many trials (e.g., 2,000).
  • Calculates probability of a rare event (e.g., more than 5200 heads) via:
    • Simulation (Monte Carlo)
    • Analytical CLT approximation
  • Visualizes outcomes with a histogram of simulation results and an overlay of the theoretical bell curve.
  • Interactive web demo: clicking "Run Simulation" computes results and updates the chart dynamically.

📊 Key Concepts Demonstrated

  • Randomness / Law of Large Numbers: multiple coin flip trials converge to predictable distribution.
  • Binomial Distribution: flipping a coin n times follows Binomial(n, p).
  • Central Limit Theorem (CLT): binomial distribution approximates normal for large n.
  • Simulation vs Theory: Monte Carlo probability vs analytical probability.
  • Rare Event Probability: extreme outcomes (e.g., >4σ) are very unlikely; simulation struggles.

🔢 Mathematical Details

  • Binomial Distribution: X ~ Binomial(n, p) with μ = n*p, σ = sqrt(n*p*(1-p))
  • CLT Normal Approximation: Z = (X - μ)/σ → P(X > k) ≈ P(Z > (k-μ)/σ)
  • Example: k = 5200, n = 10000, p = 0.5 → Z ≈ 4 → probability ≈ 0.00003
  • Simulation: P(X > k) ≈ (# of trials where heads > k) / total trials

💻 Implementation

  • JavaScript: browser simulation and probability calculation
  • Chart.js: dynamic histogram + bell curve
  • HTML + CSS: interactive front-end
  • How it works:
    • Click "Run Simulation"
    • JavaScript runs nTrials simulations of nFlips coin flips
    • Calculates simulated probability and CLT probability
    • Builds histogram of results
    • Overlays theoretical bell curve
    • Displays results and chart dynamically
